## 这是一个营销管理系统开发 
需求分析：
1、系统管理员：系统用户以及权限控制
2、管理员：负责监督客户经理的营销情况
3、客户经理：负责客户营销，实现客户营销信息录入以及精度更新、分时段查询报表
框架：VUE+element-ui+echart
